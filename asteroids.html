What is it?

Simply put it's asteroids... looks easy enough. I won't say it was. This project involed working with webGL programming - writing shaders, manipulating vertexes , all that fun stuff. I've done a bunch of development around a few game engines, including some that used webGL. This was my first real go at really getting "under the hood" of the whole video game thing and writing the actual graphics code - we all start small. :)

Have a play, have a look at the code

W,A,S,D to rotate and move, space to fire
